Xcode 常用快捷键

Dec 11, 2018

Contents

  1. 键位
  2. 快捷键
    1. 运行 ⌘ + R
    2. 编译 ⌘ + B
    3. 停止 ⌘ + .
    4. 当前文件查找 ⌘ + F
    5. 查找下一处 ⌘ + G
    6. 查找上一处 ⌘ + ⇧ + G
    7. 全局查找 ⌘ + ⇧ + F
    8. 快速查找并跳转到类或方法 ⌘ + ⇧ + O
    9. 清除编译文件夹缓存 ⌘ + ⇧ + K
    10. 在当前文件跳转到行数 ⌘ + L
    11. 清空日志 ⌘ + K
    12. 在当前文件跳转到行数 ⌘ + L
    13. 定位当前文件在左边项目目录中的位置 ⌘ + ⇧ + J
    14. .h 与 .m 文件切换 ⌘ + ⌃ + ↑ (↓)
    15. 撤销 ⌘ + Z
    16. 反撤销 ⌘ + ⇧ + Z
    17. 选中的代码向左位移 ⌘ + [
    18. 选中的代码向右位移 ⌘ + ]
    19. 选中的代码向上位移 ⌘ + ⌥ + [
    20. 选中的代码向下位移 ⌘ + ⌥ + ]
    21. 向上切换运行设备 ⌘ + ⌥ + ⌃ + [
    22. 向下切换运行设备 ⌘ + ⌥ + ⌃ + ]
    23. 选中的代码自动缩进 ⌃ + I
    24. 跳转到类, 方法或变量的定义代码处 ⌘ + ⌃ + J
    25. 折叠代码块 ⌘ + ⌥ + ←
    26. 展开代码块 ⌘ + ⌥ + →
    27. 返回到上个页面 ⌘ + ⌃ + ←
    28. 前进到下个页面 ⌘ + ⌃ + →
    29. 新建文件 ⌘ + N
    30. 新建工程 ⌘ + ⇧ + N
    31. 打开/关闭所有断点 ⌘ + Y
    32. 暂停时跳过此断点 ⌘ + ⌃ + Y
    33. 单步调试 F6
    34. 光标跳到文件首部 ⌘ + ↑
    35. 光标跳到文件尾部 ⌘ + ↓
    36. 光标跳到行首 ⌘ + ←
    37. 光标跳到行尾 ⌘ + →
    38. 光标跳到单词首部 ⌥ + ←
    39. 光标跳到单词尾部 ⌥ + →
    40. 左侧navigator跳转 ⌘ + 1~9
    41. 显示/隐藏左侧面板 ⌘ + 0
    42. 显示/隐藏右侧面板 ⌘ + ⌥ + 0
    43. 显示/隐藏下边控制台 ⌘ + ⇧ + Y
    44. 当前类里查找方法或变量 ⌃ + 6 并可直接输入查找关键字, 回车跳转
    45. 打开/隐藏Toolbar ⌘ + ⌥ + T
