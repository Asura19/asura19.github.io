Custom keys

If the field in JSON is not what we want:

We need to define a nested enum in our type called ‚ÄúCodingKeys‚Äù (or use a typealias with this name) that conforms to the CodingKey protocol ‚Äì Swift will automatically use this as the key type. This therefore allows you to easily customise the keys that your properties are decoded with.
